Tentang Saya

Menjalani peran sebagai software developer sekaligus mahasiswa IT membuat saya terbiasa melihat aplikasi dari sudut pandang fungsionalitas dan kenyamanan pengguna. Saya mengombinasikan keahlian backend dalam merancang RESTful API dan arsitektur database (menggunakan PHP/CodeIgniter 4 dan Java/Spring Boot) dengan kemahiran frontend menggunakan Flutter. Setiap baris kode lintas platform yang saya susun selalu didasari oleh analisis yang kuat serta penerapan prinsip Delapan Aturan Eight Golden Rules UI/UX untuk menciptakan produk digital yang berdampak nyata.
